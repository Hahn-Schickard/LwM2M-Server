# Lightweight Machine to Machine (LwM2M) Serve Implementations
## Brief description

This is a moder Cpp17 based LwM2M Implementation. It uses ASIO to manage the network communications, HaSLL as the main logging interface, PugiXML as the XML parser library and provides all of the required dependency managment throught Conan Dependency Mangment System.

## Required dependencies
* [Python 3.7](https://www.python.org/downloads/release/python-370/)
* [conan](https://docs.conan.io/en/latest/installation.html)
